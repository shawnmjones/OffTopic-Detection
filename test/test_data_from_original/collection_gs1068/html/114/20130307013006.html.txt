Полицейский в Приамурье выстрелил из травматики в несовершеннолетнего
Полицейский в Приамурье подозревается в том, что выстрелил в голову юноше из травматического оружия, возбуждено уголовное дело. По версии следствия, в минувшее воскресенье помощник участкового уполномоченного полиции по делам несовершеннолетних участвовал в мероприятиях по пресечению незаконного оборота наркотиков.На участке проселочной дороги, расположенном неподалеку от села, полицейские остановили для досмотра 17-летнего молодого человека. В один из моментов по невыясненным пока обстоятельствам прапорщик полиции выстрелил из принадлежащего ему травматического пистолета в голову молодого человека.

В Волгограде осудили экс-полицейского, сбившего мать с детьми
Сегодня Кировский районный суд Волгограда вынес обвинительный приговор бывшему следователю полиции, летом прошлого года сбившему мать с двумя детьми, один из которых погиб. Ему назначено наказание 3,5 года колонии-поселения. Кроме того, по иску прокурора в интересах ТФОМС с осужденного взыскано больше 300 тысяч рублей, потраченных на лечение матери. Экс-полицейский также заплатит семье 1,7 миллиона рублей в качестве компенсации морального вреда. Свою вину Александр Бурденко полностью признал. Но пока приговор в законную силу не вступил и может быть обжалован.
